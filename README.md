# Customer_behavior_data_analysis_project

# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior using transactional retail data to uncover insights about purchasing patterns, customer segments, and product performance.

The goal is to help businesses understand how customer demographics, product categories, discounts, and subscription behavior impact sales and customer engagement.

---

# Dataset Information

- Total Records: 3,900
- Total Columns: 18

### Key Features
- Customer Demographics: Age, Gender, Location
- Purchase Details: Item Purchased, Category, Purchase Amount
- Shopping Behavior: Discount Applied, Previous Purchases, Frequency of Purchases
- Customer Experience: Review Rating
- Logistics: Shipping Type
- Customer Status: Subscription Status

---

# Project Workflow

## 1. Data Cleaning & Preparation (Python)

Libraries used:
- pandas
- numpy

Key steps performed:

- Loaded dataset using pandas
- Checked dataset structure using `df.info()` and `df.describe()`
- Identified and handled missing values in **Review Rating**
- Filled missing ratings using **median rating per category**
- Created a new feature **age_group** using age segmentation
- Converted purchase frequency into numerical days
- Removed redundant columns
- Exported cleaned dataset for SQL analysis

Example code:

```python
df['Review Rating'] = df.groupby('Category')['Review Rating'].transform(lambda x: x.fillna(x.median()))
```

---

# 2. SQL Business Analysis

The cleaned dataset was imported into **MySQL** for analytical queries.

### Business Questions Answered

1. Total revenue generated by male vs female customers
2. Customers who used discounts but still spent above average
3. Top 5 highest-rated products
4. Purchase comparison between Standard vs Express shipping
5. Subscriber vs non-subscriber spending behavior
6. Products with the highest discount dependency
7. Customer segmentation (New, Returning, Loyal)
8. Top 3 most purchased products per category
9. Relationship between repeat buyers and subscriptions
10. Revenue contribution by age group

Example Query:

```sql
SELECT gender, SUM(purchase_amount) AS revenue
FROM customer_data
GROUP BY gender;
```

Advanced SQL techniques used:

- Aggregations
- Subqueries
- Window Functions
- CTEs

# Key Insights

- Certain product categories generate significantly higher revenue.
- Loyal customers contribute a major portion of repeat purchases.
- Some products depend heavily on discounts to drive sales.
- Younger and middle-aged groups generate a large portion of revenue.
- Subscription customers tend to have higher engagement levels.
